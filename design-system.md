# Design System do portfólio

## 1. Introdução
Esta é uma página de autenticação simples e minimalista.

## 2. Fundações
- Paleta de Cores:
  --text: #f3f0f1;
  --background: #0d0b0b;
  --primary: #c1b2b8;
  --secondary: #46564d;
  --accent: #8b9aa1;

- Tipografia:
    - Titulos: Inter (via Google Fonts)
    - Texto normal: Sans-serif

- Espaçamento:
  - Grande (8 ou 16px)

## 3. Componentes
- Botões:
  - Todos os botões vão:
    - Ter uma sombra pequena na parte inferior
    - Terão arredondamento de 4px
    - Terão fonte negrito
    - Seguirão o espaçamento definido na fundação.
  - Variações:
    - Primário: Seguirão a paleta a risca.
    - Secundário: Terá o valor invertido do fundo/texto.

- Campo de texto (input):
  - Todos os inputs vão:
    - Terão arredondamento de 4px
    - Seguirão espaçamento de finido na fundação

## 4. Guia de uso
- No CSS sempre usar variáveis para cores e elementos da fundação ao invés de espalhar pelo código.
- Manter a página sempre com foco principal em ter os elementos centralizados.
